Calculadora de IP para IPv4

A Calculadora de IP é um projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO). Seu objetivo é analisar um endereço IP e sua máscara de sub-rede, fornecendo informações detalhadas como o endereço de rede, primeiro e último host, endereço de broadcast, classe do IP, número de sub-redes, quantidade de hosts por sub-rede, tipo do IP (público ou privado) e intervalo de IPs disponíveis. A aplicação conta com uma interface gráfica interativa desenvolvida com Streamlit, permitindo que o usuário insira um endereço IP e uma máscara de sub-rede para visualizar os cálculos de forma clara e intuitiva.
